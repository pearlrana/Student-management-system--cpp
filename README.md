# Student Management System (C++)

This is a menu-driven application built using Object-Oriented Programming concepts in C++.

## Features
- Add student records
- Display all students
- Search student by ID
- Update student details
- Delete student
- Calculate average marks
- Save and load data using file handling

## Concepts Used
- Classes and Objects
- Encapsulation
- Constructors
- File Handling
- STL (vector)

## How to Run
- Compile using any C++ compiler
- Run the program and follow menu instructions
